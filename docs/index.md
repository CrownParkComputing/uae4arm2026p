## Features

- 🎮 **Gamepad Support** — Full support for external controllers
- 📱 **Touch Controls** — On-screen virtual keyboard and joystick
- 💾 **WHDLoad** — Pre-installed WHDLoad for one-click game launches
- 🖥️ **AROS ROM** — Includes AROS replacement ROM for legal emulation
- 🔄 **Floppy Bridge** — Real floppy drive support via hardware adapters

## System Requirements

- Android 7.0 (API 24) or higher
- ARM64 device (arm64-v8a)
- ~50MB storage for app + space for ROMs and ADFs

## Quick Start

1. [Download the APK](https://github.com/CrownParkComputing/uae4arm_2026/releases/latest/download/uae4arm_2026-release.apk)
2. Enable "Install from unknown sources" in Android settings
3. Open the APK to install
4. Launch and point to your Amiga ROMs and ADF files
